# CppCodes 📘🚀

A structured collection of **C++ notes and practice codes** created while learning from
LearnCpp.com.

This repository contains implementations, commented explanations, and examples covering **almost all fundamental C++ concepts** (≈20 out of 27 lessons), excluding only advanced topics.

---

🎯 About This Repository

This repo serves as:

* Personal learning notes
* Concept revision resource
* Practice code collection
* Beginner-friendly reference
* Interview preparation material

The notes include **code snippets + inline explanations using comments**, making them easy to understand and revise.

---

📚 Topics Covered

* C++ basics (syntax, variables, data types)
* Operators and expressions
* Control flow (if/else, loops, switch)
* Functions and scope
* Arrays and strings
* Pointers and references
* Dynamic memory allocation
* Structs and enums
* Object-Oriented Programming (classes, constructors, encapsulation, etc.)
* Function overloading
* Templates (basic)
* STL introduction
* Debugging and best practices
* Misc practice programs

> Advanced C++ topics are intentionally excluded as they are not yet completed.
